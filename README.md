# -这是经典游戏《月姬PLUS+DISC》的renpy引擎重制版本

目前我已经完成了前三章
后续慢慢更新
更新后将移植至其他（psv/3ds等平台）

感谢项目：https://github.com/SonicMastr/renpy-vita
使移植到其他平台成为可能


（game文件夹）是未加密的renpy引擎工程文件，可直接通过renpy打开
推荐使用版本renpy7.4.9.2142版本开发
欢迎有兴趣的也可以继续完善该项目
